# Roadmap

### 2.5.0
- GUI version of the tool for easier navigation and configuration - no more need to use the console and type in selections, just click through the options you want!

### 2.5.1
- Add the ability to save and load optional policy selections, so you can easily apply the same configuration across multiple devices or save your preferred settings for future use.

### 2.5.2
- Add more detailed descriptions and information about each policy setting during configuration, to help users understand the impact of each option and make informed decisions about which policies to apply.

### 2.5.3
- Improve Group Policy refresh, completion waiting, and restart handling

### 2.5.4
- Add more detailed logging and status reporting.

### 2.5.5
- Add advanced policy customization, including selecting specific policies and setting custom values for supported options.

### 3.0.0
- Major release with a polished interface, documentation, and broader deployment readiness.
